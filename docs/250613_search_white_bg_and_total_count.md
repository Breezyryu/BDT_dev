# 검색 비매칭 셀 흰색 배경 + 전체 카운트 표시

## 날짜: 2025-06-13

## 변경 파일
- `DataTool_dev/DataTool_optRCD_proto_.py`

## 변경 내용

### 1. 비매칭 셀 배경 → 흰색
- **Toyo**: alpha 투명도(180) 방식 → `QColor(255, 255, 255)` 흰색 배경 + `QColor(173,181,189)` 회색 글자
- **PNE**: alpha 투명도(200) 방식 → `QColor(255, 255, 255)` 흰색 배경 + `QColor(173,181,189)` 회색 글자
- 검색 비활성 시 동작은 기존과 동일 (회색 글자만 적용)

### 2. 전체 페이지 카운트 표시
- **표시 형식**: `전체N건 (현재M건)` — 예: `12건 (3건)`
- **구현 방식**: 
  - `self._find_counts` 딕셔너리로 각 cycler별 매칭 카운트 저장
  - `self._find_last_search` 로 검색어 변경 감지 → 변경 시 카운트 초기화
  - 키: `toyo_{num}` 또는 `pne_{num}`
  - 페이지 전환 시 해당 페이지의 카운트가 업데이트되고, 전체 합산 표시
- **검색어 변경 시**: `_find_counts` 딕셔너리 초기화 → 이전 검색 결과 무효화
