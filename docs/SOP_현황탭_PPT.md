# BatteryDataTool - 현황 탭 SOP
### 충방전기 채널 현황 모니터링

---

## 📌 1. 탭 개요

| 항목 | 내용 |
|------|------|
| **탭 이름** | 현황 |
| **목적** | Toyo / PNE 충방전기의 채널별 사용 현황을 실시간 모니터링 |
| **주요 기능** | 시험실/충방전기 선택 → 채널 테이블 조회 → 필터/강조 → 데이터 저장 |

---

## 📌 2. 화면 구성

```
┌─────────────────────────────────────────────────────────┐
│ [시험실 선택▼] [충방전기 선택▼] [표시항목 선택▼]  강조할 문자 [___] │
├──────────┬──────────────────────────────────────────────┤
│ 사용 가능: N │  범례 테이블 (온도별 색상, 상태별 배경색)            │
│ 사용 중: M  │                                                  │
├──────────┴──────────────────────────────────────────────┤
│                                                          │
│              채널 테이블 (8열 × 16행)                      │
│         각 셀: [채널번호 | 선택한 표시항목 정보]              │
│                                                          │
└─────────────────────────────────────────────────────────┘
```

---

## 📌 3. 조작 순서 (SOP)

### Step 1: 시험실 선택 (`tb_room`)

| 인덱스 | 시험실 | 포함 충방전기 |
|--------|--------|--------------|
| 0 | R5 15F | Toyo1~5, PNE1~5 |
| 1 | R5 3F B-1 | PNE01~08 |
| 2 | R5 3F B-2 | PNE09~16 |
| 3 | R5 3F A | PNE17~25 |
| 4 | 전체 | 모든 충방전기 |

> **동작**: 시험실 변경 시 → 충방전기 콤보박스 목록이 자동 갱신됨

---

### Step 2: 충방전기 선택 (`tb_cycler`)

- 선택한 시험실에 따라 표시되는 충방전기 목록이 달라짐
- **Toyo 시리즈**: Toyo1 ~ Toyo5 (Z: 드라이브)
- **PNE 시리즈**: PNE1~5 (Y:/X:), PNE01~25 (W:/V:/U: 드라이브)

> **동작**: 충방전기 변경 시 → 테이블 리셋 후 해당 장비의 채널 정보 로드

---

### Step 3: 표시 항목 선택 (`tb_info`)

| 인덱스 | 항목 | 설명 |
|--------|------|------|
| 0 | 채널 | 채널 번호 + 상태 |
| 1 | 파일명 | 테스트 파일명 |
| 2 | 스케쥴명 | 스케줄(프로파일)명 |
| 3 | 날짜 | 테스트 시작 날짜 |
| 4 | 파트 | 담당 파트 |
| 5 | 이름 | 담당자 이름 |
| 6 | 온도 | 시험 온도 |
| 7 | 현재상태 | 현재 Step / Loop Cycle / 총 Cycle |
| 8 | 현재 전압 | 실시간 전압 |
| 9 | 셀 경로 | 데이터 저장 경로 |

> **동작**: 표시 항목 변경 시 → 채널 테이블이 선택 항목 기준으로 재표시

---

### Step 4: 강조 문자 필터 (`FindText`)

- 텍스트 입력란에 **특정 키워드** 입력
- 해당 키워드를 포함하는 채널만 **정상 색상**으로 표시
- 미포함 채널은 **회색(175,175,175)** 처리

---

## 📌 4. 채널 테이블 색상 범례

### 배경색 (채널 상태)

| 색상 | Toyo | PNE |
|------|------|-----|
| 🟧 주황색 (255,127,0) | 완료 / 작업정지 | 완료 (셀 있음) |
| 🟦 하늘색 (200,255,255) | 전압 없음 (사용 가능) | 대기 / 준비 (셀 없음) |
| 🟪 분홍색 (255,200,229) | - | 작업멈춤 |
| ⬜ 회색 (200,200,200) | - | 강조 문자 필터 해당 |

### 글자색 (시험 온도)

| 색상 | 온도 범위 |
|------|----------|
| 🔵 파란색 (0,0,255) | 15°C (10 < T ≤ 20) |
| ⚫ 검은색 (0,0,0) | 23°C (기본) |
| 🟢 녹색 (0,255,0) | 35°C (30 < T ≤ 40) |
| 🔴 빨간색 (255,0,0) | 45°C (40 < T ≤ 50) |

## 📌 6. 데이터 흐름

```
시험실 선택
    │
    ▼
충방전기 목록 갱신
    │
    ▼
충방전기 선택
    │
    ├─ Toyo? ──► Z:\Working\BLKn\Chpatrn.cfg 읽기
    │              + ExperimentStatusReport.dat 읽기
    │
    └─ PNE?  ──► Y/X/W/V/U:\Working\PNEn\Module_n_channel_info.json 읽기
    │
    ▼
DataFrame 생성 (채널번호, 상태, 테스트명, 온도, 사이클, 전압 등)
    │
    ▼
tb_channel 테이블에 채널별 정보 표시
    │
    ├─ 배경색: 채널 상태별 (완료/대기/멈춤)
    ├─ 글자색: 시험 온도별 (15/23/35/45°C)
    └─ 강조 필터: FindText 키워드 매칭
```

## 📌 8. 주의사항

1. 네트워크 드라이브가 마운트되어 있어야 정상 동작
2. Toyo4는 다른 Toyo와 채널 구성이 다름 (5×2 = 10ch)
3. PNE는 Module_1/Module_2 JSON을 합쳐서 읽음
4. 전압 0.04V 이하는 "-"로 표시 (미접속/미사용)
5. 강조 문자는 **testname** 필드 기준으로 필터링
