# Phase 2 — 신 9종 UI 색상표 + 타임라인 툴팁 한글화

**날짜**: 2026-04-19
**대상 파일**: `DataTool_dev_code/DataTool_optRCD_proto_.py`
**선행**: [Phase 1 변경로그](260419_사이클분류_Phase1_proto이식.md)

---

## 목적

Phase 1 에서 신 9종 + 서브태그 체계를 분류기에 이식했으나, `_CLASSIFIED_COLORS` 는 구 6종 구조만 있어 새 카테고리(예: `저장(floating)`, `SOC별 사이클`)가 기본색·기본 라벨로 표시될 위험이 있었음. 이번 단계에서 **색상표를 신 21키로 확장** + **타임라인 툴팁/상세 팝업이 자동으로 한글 라벨 표시**하도록 마감.

---

## 변경 내용

### 1. `_CLASSIFIED_COLORS` 확장 (L8128)

기존 7키 → **27키** (신 21 + 구 약어 alias 6). `THEME['PALETTE']` 의 10색을 의미별로 일관되게 배분:

| color_idx | hex | 카테고리 계열 |
|:---:|:---:|---|
| 0 | `#3C5488` 네이비 | 사이클 (ACCEL / FORMATION / 통합) |
| 1 | `#E64B35` 빨강 | DCIR (+ 구 Rss alias) |
| 2 | `#00A087` 청녹 | 충전 (세팅) |
| 3 | `#F39B7F` 연주황 | GITT (full / simplified) |
| 4 | `#4DBBD5` 하늘 | 방전 (초기 / 종료 / 저장준비 / SOC세팅) |
| 5 | `#8491B4` 회보라 | RPT |
| 6 | `#B09C85` 베이지 | SOC별 사이클 |
| 7 | `#91D1C2` 민트 | 저장 (floating 포함) |
| 8 | `#DC0000` 진빨강 | 히스테리시스 (충전 / 방전) |
| 9 | `#7E6148` 브라운 | unknown |

**서브태그는 주 카테고리와 같은 색상을 공유** — 시각 일관성 유지. 서브태그 구분은 툴팁/상세 팝업의 한글 텍스트로 제공.

**구 약어(Rss / 가속수명 / initial / 반사이클 / unknown) 도 그대로 유지** — 기존 색상 불변, 하위 호환.

### 2. 타임라인 툴팁 한글화 — **코드 변경 없이 자동 적용**

기존 `CycleTimelineBar` 는 이미 `pat['desc']` 를 hover 툴팁·상세 팝업에 사용:

- **Hover 툴팁** (L8745): `"사이클 {start}~{end} ({count}cy)\n{pat['desc']}"`
- **상세 팝업 블록 라벨** (L23221): `name = pat['desc']`
- **우클릭 컨텍스트 메뉴** (L8780): `f"이 블록 선택 ({pat_name} {start}-{end})"`

`_CLASSIFIED_COLORS` 의 `desc` 필드를 한글로 업데이트했으므로 **위 3개 지점 모두 자동으로 한글 라벨 표시**. UI 코드 별도 수정 불필요.
