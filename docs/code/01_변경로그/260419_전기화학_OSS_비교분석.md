## 핵심 발견

1. **PyBaMM = OSS P2D 표준 (BSD-3, 생태계 최대)** — 그대로 유지.
2. **PETLION.jl = 속도 챔피언**: 2.96 ms/discharge, LIONSIMBA 대비 206×, PyBaMM 대비 35×.
3. **BattMo.jl = 가장 광범위한 multiphysics + adjoint AD**, 1D/2D/3D 통합 코드 경로.
4. **PyBOP = PyBaMM 생태계 유일 본격 estimation 프레임워크** (25+ optimizer × 17+ PINTS MCMC).
5. **COBRAPRO**만 DFN에 식별성 분석(LSA+상관)을 통합.
6. **PINNSTRIPES = BDT Bayesian 역추정의 참조 구현** (BSD-3, self-adaptive + annealing + curriculum + hard-IC + NumPyro).
7. **moleculer-db 오인 링크 확인** — JavaScript microservice DB 모듈, 배터리 MD와 무관. 실제 배터리 MD는 LAMMPS/GROMACS/OpenMM 계열.
