- ⚠️ **#7, #8 ACCEL → SOC별 사이클 오분류** — Rule 1 (`EC≥4 + N≥5 → SOC별 사이클`) 이 너무 광범위. 가속수명에도 EC(용량 종료조건)가 있음.

### 4.1 Rule 1 보강 제안
**SOC별 사이클** 판별에서 "EC 개수" 뿐 아니라 **EC 값의 다양성(여러 SOC 레벨)** 확인 필요:

```python
# 현재
if N >= 5 and n_ec >= 4 and n_steps >= 8:
    return ('SOC별 사이클', None)

# 개선안
ec_values = {s.get('ec_value', 0) for s in ec_steps}
if N >= 5 and n_ec >= 4 and n_steps >= 8 and len(ec_values) >= 3:
    return ('SOC별 사이클', None)
# EC 값이 3개 이상 다른 SOC 를 훑을 때만 SOC별 사이클
# 단일 값 반복(같은 DOD 에서 반복 ACCEL)은 사이클(ACCEL) 로 넘김
```

Phase 1 이식 전 이 보강 반영 필수.
