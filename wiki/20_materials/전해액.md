# 개요

리튬 이온 배터리 전해액 허브. 기본 구성 (용매 · 리튬 염 · 첨가제) + 분석 방법 + 조성 DOE + 코인셀 부반응 실험.

## 기본 구성

- **용매**: EC, DMC, EMC, PC 등 카보네이트 / 에스테르 계열
- **리튬 염**: LiPF₆, LiTFSI, LiBF₄ 등
- **첨가제**: FEC, VEC, PS, SN, HTCN, LiPO₂F₂, LiBF₄, LiDFOB 등

물성 요구 조건: 이온 전도도, 산화/환원 안정성, 점도.

# 전해액 조성 (DOE)

- **DOE1**: 1.35M LiPF₆ / EC:PC:EMC=15:15:70 + 7%FEC + 3%PS + 0.5%LiPO₂F₂
- **DOE2**: 1M LiPF₆ / EC:EMC=25:75 / SN 3%
- **DOE3**: 1M LiPF₆ / EC:EMC=25:75 / HTCN 3%
- **DOE4**: 1M LiPF₆ / EC:EMC=25:75 / SN 2%, HTCN 2%
- **Ref1**: 1.35M LiPF₆ / EC:PC:EP:PP=15:15:30:40 + 7%FEC + 1%VEC + 0.2%LiBF₄ + 3%PS + 3%HTCN + 2%SN + 0.3%LiPO₂F₂ + 1%LiDFOB
- **Ref2**: 1.35M LiPF₆ / EC:PC:EMC=15:15:70 + 7%FEC + 1%VEC + 0.2%LiBF₄ + 3%PS + 3%HTCN + 2%SN + 0.3%LiPO₂F₂ + 1%LiDFOB
