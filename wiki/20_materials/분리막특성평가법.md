| 요구 특성                      |                    | 의미                                 | 요구 수치         | 규격         | 평가 장비 및 방법                              |
|----------------------------|--------------------|------------------------------------|---------------|------------|-----------------------------------------|
| Thickness (㎛)              |                    | 두께↓ = 전지 Capa.↑                    | < 25          | ASTM D5947 | 미소 두께 측정기 or SEM                        |
|                            |                    |                                    |               | ASTM D2103 |                                         |
| Pore size (㎚)              |                    | Size 분포도가 균일할 수록 유리                | < 100         | ASTM E128  | * PMI(Capillary Flow Porometer)         |
|                            |                    |                                    |               | ASTM 1294  |  :  낮은 표면 E를 갖는 액체로 기공을 채운 후            |
|                            |                    |                                    |               |            |   압력을 가해 다시 기공을 비워가며 size 측정            |
|                            |                    |                                    |               |            | * Hg porometer                          |
|                            |                    |                                    |               |            |  : Pore size ∝ 1/Pressure               |
| Porosity (%)               |                    | 전해질 보유량, 기계적 강도                    | > 40          | ASTM D1622 | * Hg porometer                          |
|                            |                    | (Porosity = 빈 공간의 부피/전체부피)         |               |            |  : 높은 압력 하에 기공에 투입된 수은 부피 측정            |
|                            |                    |                                    |               |            |  : Porosity ∝ Volume of mercury         |
| 투과도                        | 공기투과도              | 분리막에 대한 공기 투과도                     | < 300/100mil  | ASTM D726  | * PMI(Capillary Flow Porometer)         |
|                            | (Gurley value, s)  | 이온 투과 능력에 대한 지표                    |               |            |  : 규정 압력하에 주어진 양의 공기가 일정한 면적을           |
|                            |                    |                                    |               |            |    통과하는데 걸리는 시간                         |
|                            | 전기저항도              | 전기 저항값으로 이온의 이동도                   | < 8           | -          | MN=분리막이 장착된 상태의 저항/전해질만의 저항             |
|                            | (Macmullin number) | 간접 비교                              |               |            |                                         |
| Tensile strength (kgf/cm2) |                    | 분리막의 인장 강도                         | > 1,500       | ASTM D882  | * Universal testing machine             |
|                            |                    |                                    | (MD, TD)      |            |  : 인장 하중을 받을 때 하중과 연신의 관계               |
|                            |                    |                                    |               |            |  : 기준면적: width * thickness=1 in. * 20μm |
| Puncture strength (g/mil)  |                    | 이물질에 찔렸을 때의 안정성                    | > 500         | ASTM D3763 | * 인장 시험기에 일정 크기의 송곳을 장착하고, 분리막에         |
|                            |                    |                                    |               |            |   일정 속도로 낙하시켜 그 파단 강도를 측정               |
|                            |                    |                                    |               |            |  : 기준 면적: 90mm(diameter) 이상             |
| Shrinkage (%)              |                    | 과열상태에서의 내구성 (폭발 방지)                | < 5% (MD, TD) | ASTM D1204 | * 90℃에서 1시간 건조 후 MD, TD 방향의 수축률 측정      |
|                            |                    |                                    |               |            |  : 기준면적: 25cm * 25cm                    |
|                            |                    |                                    |               |            |  : Linear change % = [(Df-D0)/D0]*100   |
| Wettability                |                    | 전해액과의 친화성                          |               | -          | 접촉각                                     |
| Chemical stability         |                    | 산화, 환원 조건 하의 화학적 안정성, 전해질에 대한 내부식성 |               | -          |                                         |
| Shut down Temp.            |                    | 과열시 단락 방지 성능                       | < 125℃        | -          | * 볼트셀 법                                 |
|                            |                    |                                    | (낮을수록 유리)     |            | : AC impedence spectroscopy             |
| Melt down Temp.            |                    | 고온에서의 내열 특성                        | > 150℃        | -          | : 온도 상승시키며 임피던스 측정                      |
|                            |                    |                                    | (높을수록 유리)     |
