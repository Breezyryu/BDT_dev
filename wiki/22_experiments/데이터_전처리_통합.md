# 1. Overview

## 1.1 경로 입력 및 정리
- 경로입력
- 경로 정리: `/`, `\` 구분자 정규화, 따옴표 제거
- 경로에서 용량정보 추출 (`capacity_min` double형)

## 1.2 충방전기 제조사 구분
- **PNE vs Toyo**: `Pattern` 폴더 유무로 판단
  - Pattern 폴더 존재 → PNE
  - 없음 → Toyo

## 1.3 기타 전처리 파라미터
- 양극 Capacity loading
- 음극 NP ratio
- 양극 활물질 up → 양극 비용량 감소

## 1.4 시간 스케일 메모 (예시)
- 6:40
- 2h 37m → 5h 30m
- 6:45 3h
- 5:00 1h20m
- 4:00 20m
- 5:40 2h
