## 4. 운영 흐름도 (2025-04-17)

> 원본: `2025-04-17_글로브박스.md` (Excalidraw). 아래는 텍스트 요소 정리본.

### 4.1 해체분석 흐름
- **활성화셀**: 포항공대
- **오픈셀**: 카이스트
- **분해/분석 시점**: (HHP/Tablet) 400, 800, 1000, 1200 cycle

### 4.2 분석
- **코인셀 (Full / Half)**
  - EP/PP → lithium metal 반응성으로 half 셀 충방전 특성 안 좋음
  - Half cell Electrolyte: 1M LiPF6 EC/DEC (3:7 vol%)
  - Half cell Current Density: 0.2 mA/cm²
- **EIS**
- **단면분석**: CP (Cross-section Polishing) + SEM
- **DMC 함침 → Rinsing → Punching**

### 4.3 전해액 조성
**Ref1**: 1.35M LiPF6 / EC:PC:EP:PP = 15:15:30:40 + 7% FEC + 1% VEC + 0.2% LiBF4 + 3% PS + 3% HTCN + 2% SN + 0.3% LiPO2F2 + 1% LiDFOB

**Ref2**: 1.35M LiPF6 / EC:PC:EMC = 15:15:70 + 7% FEC + 1% VEC + 0.2% LiBF4 + 3% PS + 3% HTCN + 2% SN + 0.3% LiPO2F2 + 1% LiDFOB

**Ref3**: 1.35M LiPF6 / EC:PC:EMC = 15:15:70 + 7% FEC + 3% PS + 0.5% LiPO2F2

**기타**: 1.35M EC/EMC + 7wt% FEC

참고: [[전해액 분석 및 조성]]

### 4.4 글로브박스 관리/점검
- **Ar 가스**: 1~2개월 교체 주기
- **Purification material**: ~10개월

### 4.5 소모품
- **트위저**
  - 분리막용: 고정밀 평 핀셋 (Ideal Tek / 33A.SA)
  - 전극용: 세라믹 팁
- **쿠킹호일 / 킴텍스**
- **보관용기**
  - Vial (~30 mL): Borosilicate glass
  - Petri dish (Ø35 × H10 mm)
- **유산지** (펀칭용 및 무게 측정): Weighing Dish
- **Parafilm**
- **블로우**
- **글러브**: Poly Gloves 폴리글로브 (S, M)
  - Carbon/fiber: 내화학성 O / 전도성 X
  - 세라믹/PPS/PEEK: 내화학성, 절연성, 내열성

### 4.6 화학물질 (사전평가 및 유해화학물질 관리 대상, 5년 보관)
- NMP
- DMC
- 에탄올 (무수화)
- 전해액 (앤켐)
- 리튬메탈칩

### 4.7 관련 장비
- **코인셀**
  - 펀칭기 (Ø14, Ø16, Ø19) — 시노프로
  - Ø19: 분리막용 / Ø16: 음극, 리튬메탈 (0.3t) / Ø14: 양극
  - Clamping / Unclamping
  - Powder machine
- **파우치**
  - Vacuum Sealing machine
  - Thermal Sealing
- **삼전극 파우치**
