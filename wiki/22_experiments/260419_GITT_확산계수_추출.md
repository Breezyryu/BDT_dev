# GITT 로 고체 확산계수 D_s 추출 — 원리·유도·해석

> [!abstract] 요약
> GITT (Galvanostatic Intermittent Titration Technique) 은 **저율 정전류 펄스 + 긴 휴지** 반복으로 **고체 내 Li⁺ 확산계수 D_s** 를 SOC 별로 추출하는 기법. Weppner-Huggins 공식(1977) 에서 유도된 `D_s = (4/πτ)·(n_M V_M / S)²·(ΔE_s/ΔE_τ)²` 을 구형 입자 가정으로 단순화하면 `D_s = (4/9π)(r·dE/dt / dU/d√t)²`. 본 노트는 Fick 의 법칙부터 GITT 의 실험 설계 원리, 데이터 해석, BDT 에서의 적용 지점까지 체계적으로 정리한다.

---

## 1. Why — 왜 D_s 가 중요한가

### 1.1 배터리 설계에서 D_s 의 역할
고체 전극 내 Li⁺ 확산계수 **D_s** 는 배터리의 **율특성(rate capability)** 과 **실용 용량** 을 결정하는 근본 파라미터다.

- **C-rate 한계**: 고율 방전 시 입자 중심과 표면 사이 Li⁺ 농도구배가 발생 → 표면부터 사용 불가. `τ_diff ≈ r²/D_s` 가 pulse 시간보다 크면 용량 일부를 사용 못함.
- **Pulse power**: EV·HEV 의 가속 성능은 초 단위 pulse 중 D_s 가 좌우.
- **저온 성능**: D_s 는 Arrhenius 법칙 따라 지수적으로 감소 — 저온에서 수명과 용량 모두 급락.
- **수명 평가**: 열화된 전극은 D_s 가 감소 (SEI 성장으로 표면 확산 저항 증가) — 열화 모드 진단 지표.

### 1.2 D_s 의 일반 범위
| 재료 | D_s [m²/s] | 특이사항 |
|------|-----------|---------|
| **NMC 양극** | 10⁻¹⁰ ~ 10⁻¹² | SOC 에 따라 2~3 order 변동 |
| **LCO 양극** | 10⁻⁹ ~ 10⁻¹¹ | 50% SOC 근처 최대 |
| **흑연 음극** | 10⁻¹¹ ~ 10⁻¹³ | 상전이 구간(stage 변환) 에서 급락 |
| **실리콘** | 10⁻¹⁴ ~ 10⁻¹⁶ | 고용량 but 느림, 복합 필요 |
| **LFP 양극** | 10⁻¹⁷ ~ 10⁻¹⁸ | 2상 공존 — Fickian 모델 부적합 |

→ **D_s 의 SOC 의존성을 맵핑**하는 것이 전극 최적화의 핵심. 이것이 GITT 의 주 목적.

### 1.3 BDT 관점
BDT 는 현재 **GITT 프로파일 시각화** 만 제공 (voltage curve, dQ/dV). D_s 정량 추출은 수기. F1 차용 시 **자동 SOC-별 D_s 테이블 + 플롯** 생성 → 배터리 소재 평가 가속.

## 2. What — 물리·수학 원리

### 2.1 고체 내 확산 현상

**Li⁺ 의 고체 내 이동 경로**:
1. 입자 표면 반응: Li⁺ + e⁻ + Host ⇌ LiHost (interfacial charge transfer)
2. 입자 내부 확산: 호스트 격자를 통한 hopping (diffusion)
3. 이 두 단계가 직렬 → 느린 쪽이 전체 속도 결정

**확산 = Brownian motion 의 연속체 극한**. 원자 스케일에서는:
- Li⁺ 이 격자점 사이 뛰어넘음 (hopping)
- 활성화 에너지 E_a (~ 0.3-0.5 eV for graphite)
- 거시적 D_s = (1/6)·λ²·ν·exp(-E_a/kT) — Einstein-Arrhenius

### 2.2 Fick 의 법칙

**제1 법칙** (정상상태 flux):
$$J = -D_s \cdot \nabla c$$
- J: Li⁺ flux [mol/m²/s]
- c: 농도 [mol/m³]
- ∇c: 농도 구배

**제2 법칙** (시간 진화):
$$\frac{\partial c}{\partial t} = D_s \cdot \nabla^2 c$$

구형 대칭(1D radial):
$$\frac{\partial c}{\partial t} = D_s \cdot \frac{1}{r^2}\frac{\partial}{\partial r}\left(r^2 \frac{\partial c}{\partial r}\right)$$

### 2.3 구형 입자 + Galvanostatic 경계조건

GITT 실험은 **입자 표면에 일정 flux J₀** 를 부여:
- 경계: `−D_s·∂c/∂r |_{r=R} = J₀ = I / (n·F·A_s)` (I=전류, A_s=활물질 표면적)
- 중심: `∂c/∂r |_{r=0} = 0` (대칭)
- 초기: `c(r, 0) = c_∞` (균일 평형)

이 경계값 문제의 해는 복잡한 급수지만, **짧은 시간 극한** (t ≪ R²/D_s) 에서 입자 표면은 **무한 반평면(semi-infinite)** 처럼 거동:

### 2.4 Semi-infinite 근사 — Sand 방정식

반평면 x>0 에 t=0 부터 일정 flux 부여 시 농도 변화 (오차함수):
$$c(x,t) - c_\infty = \frac{2 J_0 \sqrt{D_s t / \pi}}{D_s}\cdot\text{erfc}\left(\frac{x}{2\sqrt{D_s t}}\right)$$

표면(x=0) 농도변화:
$$\Delta c_s(t) = \frac{2 J_0}{\sqrt{\pi}}\cdot\sqrt{\frac{t}{D_s}}$$

즉 **표면 농도가 √t 에 비례해 증가** — 이것이 GITT 의 핵심 관찰량.

### 2.5 전위 → 농도 연결

표면 농도 Δc_s 가 **OCV 변화 ΔE** 로 연결되는 방식:
$$\Delta E = \frac{\partial E}{\partial c}\Big|_{eq} \cdot \Delta c_s$$

OCV-SOC 곡선의 기울기 `∂E/∂c` 는 측정 가능.

종합:
$$\Delta E(t) = \underbrace{\frac{\partial E}{\partial c}}_{\text{OCV slope}} \cdot \underbrace{\frac{2 J_0}{\sqrt{\pi D_s}}}_{\text{Sand}}\cdot \sqrt{t}$$

→ **ΔE vs √t 가 선형**. 기울기 `m = dU/d√t` 를 회귀로 추출.

### 2.6 Weppner-Huggins 공식 (1977)

Weppner & Huggins 가 위 관계를 **박막전극 (평면 기하)** 에 대해 정리:

$$\boxed{D_s = \frac{4}{\pi\tau}\left(\frac{n_M V_M}{S}\right)^2\left(\frac{\Delta E_s}{\Delta E_\tau}\right)^2}$$

- τ: pulse 지속시간
- n_M V_M / S: 전극 몰부피 / 표면적 (확산 길이 스케일)
- ΔE_s: pulse 후 최종 E_eq 변화 (해당 pulse 가 유발한 SOC 변화에 해당)
- ΔE_τ: pulse 중 voltage 변화 (= `m·√τ`, 즉 Sand 기울기 × √τ)

### 2.7 구형 입자 기하 단순화

활물질 입자를 **반지름 r 의 구**로 보면:
$$\frac{n_M V_M}{S} = \frac{(4/3)\pi r^3}{4\pi r^2} = \frac{r}{3}$$

대입하고 `ΔE_s/τ ≡ dE/dt` (pulse 간 OCV 변화율), `ΔE_τ/√τ ≡ dU/d√t` (pulse 내 Sand 기울기):

$$\boxed{D_s = \frac{4}{9\pi}\left(r\cdot \frac{dE/dt}{dU/d\sqrt{t}}\right)^2}$$

이것이 **ampworks 및 대부분의 실용 GITT 도구가 쓰는 형태**.

**물리적 해석**:
- `dU/d√t` 큼 ⟺ 표면 농도 축적 빠름 ⟺ 확산 느림 ⟺ D_s 작음
- `dE/dt` 큼 ⟺ OCV 곡선 가파름 ⟺ 분자 커짐 ⟺ D_s 큼 (같은 |m| 일 때 더 많은 SOC 변화 의미)

## 3. How — GITT 프로토콜 설계 원리

### 3.1 프로토콜 구조
```
[평형 OCV_0] ── I 펄스 τ_p ─► [voltage drop] ── Rest τ_r ─► [E_eq,1]
                                                                │
                [같은 패턴 반복]                                 ▼
[E_eq,1] ── I 펄스 τ_p ─► [voltage drop] ── Rest τ_r ─► [E_eq,2]
...
```

### 3.2 설계 제약

**① Pulse 시간 τ_p**:
- 하한: 표면 확산 response 발현 (~5 s)
- 상한: semi-infinite 근사 유효 `τ_p ≪ r²/D_s` ⇒ 10~30 min 이내 (NMC 양극 5μm, D_s=10⁻¹¹: r²/D_s ~40 min)

**② Rest 시간 τ_r**:
- 평형 도달까지 충분 ⇒ `τ_r ≫ r²/D_s` ⇒ 1~4 h

**③ ΔSOC per pulse**:
- OCV 곡선의 **국소 선형 근사** 가정 → ΔSOC ≤ 5% 권장
- 전류: C/20 ~ C/10 × τ_p 이면 적절

**④ 전압 cutoff**:
- ΔE_τ 가 너무 크면 linearization 파괴
- ΔE_τ 를 20~50 mV 유지

### 3.3 가정과 한계

Weppner-Huggins 공식 **성립 조건**:
1. **Fickian diffusion 유효** — 단일상 고용체 (solid solution). LFP 같은 2상 시스템은 부적합
2. **구형 단일입자** — 실제는 2차 입자 (1차 입자 응집체), 기공 구조 무시
3. **Semi-infinite 근사** — τ_p 가 확산 시간 스케일보다 짧음
4. **국소 선형 OCV** — pulse 중 dU/dx 일정
5. **Ohmic·charge-transfer 저항 무시** — pulse 직후 voltage drop 은 제외해야 (iR, η_ct)

**실전 보정**:
- IR drop 자동 제외: pulse 시작 첫 1~2s 회귀 제외
- 2차 입자 보정: r 에 1차 입자 반경 사용 (일반적), 또는 effective r 사용

## 4. Practice — 데이터 해석 실전

### 4.1 √t vs V 회귀 — 이상적 그래프

```
V [V]
 ▲
 │  pulse 시작 직후 iR jump
 │   ▼
 │   ●━━━━━━━━━━━━  기울기 m = dU/d√t
 │   │╲            ↑ 이 구간 [1s, 60s] 회귀
 │   │  ╲━━━
 │   │      ╲━━
 │   │        ●
 └───┴─────────────► √t  [s^0.5]
    √t=1    √t=√60
```

- 기울기 m: Sand 이론 예측 기울기
- 절편 b (+ iR 보정): pulse 직후 "유효 starting voltage"

### 4.2 SOC-별 D_s 맵핑

```
D_s [m²/s]
10⁻⁹ ┤                         
     │       ╱╲                
10⁻¹⁰┤      ╱  ╲       ╱─╲    ← 전형적 양극 NMC
     │     ╱    ╲     ╱   ╲   
10⁻¹¹┤────╯      ╲───╯     ╲  
     │                      ╲ 
10⁻¹²┤                       ╲
     └────────────────────────► SOC
     0%    25%   50%   75%   100%
     
특징:
  - 0-5%, 95-100% 에서 급락 (상변화/포화)
  - 중간 SOC 영역은 ~flat
  - 두 전극을 동시 측정하면 합본 곡선의 valley 발견
```

### 4.3 해석 체크리스트

✅ **정상 데이터 시그니처**:
- √t 회귀 R² > 0.99
- D_s 가 10⁻¹⁵ ~ 10⁻⁹ m²/s 물리적 범위
- SOC 양끝에서 값이 감소 (수십 배)
- pulse 간 dE/dt 부호 일관 (단방향 시험)

❌ **이상 시그니처**:
- D_s > 10⁻⁸: iR 미제거 또는 회귀 구간 오류
- D_s < 10⁻¹⁸: 2상 물질 (LFP) or 측정 노이즈
- SOC-D_s 곡선에 랜덤 spike: 펄스 경계 검출 오류
- 양극/음극 D_s 값 반대 크기: r 입력 오류 (5μm ↔ 12μm 혼동)

## 6. GITT 대안·관련 기법

| 기법 | 원리 | 장단점 |
|------|------|--------|
| **GITT** | galvanostatic pulse + rest | ✅ 단순, 직관적 ❌ 시간 오래 |
| **PITT** (Potentiostatic ITT) | voltage step + current decay | ✅ 빠름 ❌ 장비 제약 |
| **EIS** | AC 주파수 스윕 | ✅ 전 저항 성분 분리 ❌ 고가 장비 |
| **CITT** (Current-ITT) | ramping current | 실험 빠름, 해석 복잡 |
| **ICI** ([[ACIR_DCIR_RSS]] 참조) | rest phase 회귀 | GITT 변형, 짧은 rest |

BDT 현재 범위: GITT + ICI (simplified GITT). PITT/EIS 는 BDT 범위 밖 (별도 장비).

## 7. 상세 수식 유도 (부록)

### 7.1 Sand 방정식 상세 유도

**문제 설정**: 반평면 x > 0, t < 0 에 c = c_∞ 평형. t = 0 부터 경계 x=0 에 일정 flux J_0 부여.

**Fick 2nd law** (1D):
$$\frac{\partial c}{\partial t} = D_s \frac{\partial^2 c}{\partial x^2}$$

Laplace 변환 `L{c}(x,s) = C̄(x,s)`:
$$s C̄ - c_\infty = D_s \frac{d^2 C̄}{dx^2}$$

일반해: `C̄ = c_∞/s + A·exp(-√(s/D)·x)` (x→∞ 유한성)

경계조건: `-D_s·∂c/∂x = J_0` at x=0 (모든 t>0)
Laplace: `-D_s·(-√(s/D)·A) = J_0/s` → `A = J_0 / (s√(sD_s))`

표면 (x=0):
$$\bar{C}(0,s) - \frac{c_\infty}{s} = \frac{J_0}{s\sqrt{sD_s}} = \frac{J_0}{D_s^{1/2}}\cdot s^{-3/2}$$

역변환 `L⁻¹{s^(-3/2)} = 2√(t/π)`:
$$\Delta c(0, t) = \frac{2 J_0}{\sqrt{\pi D_s}}\cdot \sqrt{t}$$

### 7.2 Weppner-Huggins 공식 도출

Flux J_0 와 전류 I 관계:
$$J_0 = \frac{I}{n F A_s}$$

ΔE 와 Δc 관계 (국소 선형):
$$\Delta E_\tau = \frac{dE}{dc}\cdot \Delta c_s(\tau) = \frac{dE}{dc}\cdot \frac{2 I}{n F A_s \sqrt{\pi D_s}}\sqrt{\tau}$$

Pulse 총 SOC 변화 ΔE_s (쿨롱 적분):
$$\Delta c_{total} = \frac{I\tau}{n F V_M} \Rightarrow \Delta E_s = \frac{dE}{dc}\cdot \frac{I\tau}{n F V_M}$$

두 식에서 `dE/dc` 소거하고 재정리:
$$\frac{\Delta E_\tau}{\Delta E_s} = \frac{2 V_M \sqrt{\tau}}{A_s \sqrt{\pi D_s}}$$

제곱하고 D_s 에 대해 풀면:
$$D_s = \frac{4 V_M^2 \tau}{\pi A_s^2}\cdot \left(\frac{\Delta E_\tau}{\Delta E_s}\right)^{-2} = \frac{4}{\pi \tau}\left(\frac{V_M}{A_s}\right)^2\left(\frac{\Delta E_s}{\Delta E_\tau}\right)^2$$

이게 Weppner-Huggins (1977). 몰당 부피/표면적 비율에 n_M (전극 내 활물질 몰수) 곱하여 무차원 기하 인자로 재정리하면 noted 형태가 된다.
