> [!note] wiki copy
> 8. LLVM/Clang windows 설치
> [https://www.vompressor.com/clang-windws/를](https://www.vompressor.com/clang-windws/%EB%A5%BC) 참조
> ----> [https://releases.llvm.org/download.html에서](https://releases.llvm.org/download.html%EC%97%90%EC%84%9C) 원하는 버전을 설치함.
> ----> 예를 들어, [https://github.com/llvm/llvm-project/releases/tag/llvmorg-17.0.4](https://github.com/llvm/llvm-project/releases/tag/llvmorg-17.0.4) 에서 LLVM-17.0.5-win64.exe를 설치
> ----> "Add LLVM to the system PATH for all users"를 선택
