# 배터리 시뮬레이션 용어사전

> **재구조화 원칙 (2026-04-22, v3 확장 2026-04-28)**: 평면 나열(P2D/구조/열) 대신 **"시뮬레이션이 필요로 하는 코어 개념"** 16개를 1차 분류로 한다.
> 각 개념 안에는 **① 시뮬 용어** (영문·약어·단위·정의) 가 메인, **② 관련 배터리 도메인 용어** 가 부속 설명, **③ 측정·검증 anchor** 가 적용 경로로 묶인다.
>
> 일반 배터리 용어는 [[!용어]] 참조. 본 사전은 **시뮬 관점**에 한정한다.

# 1. 셀 모델 계층 (Cell Model Hierarchy)

> **개념**: 시뮬은 어떤 추상화 수준을 택할 것인가? 모델 계층이 정확도-속도 trade-off 와 다룰 수 있는 물리를 결정한다.

## 1.1 시뮬 용어

| 한글 / 영문 | 약어 | 정의 |
|----------|:----:|------|
| 단일 입자 모델 / Single Particle Model | **SPM** | 각 전극을 대표 입자 1개로. 전해질 균일 가정. 가장 빠름 (BMS 실시간). |
| SPM + 전해질 / SPM with electrolyte | **SPMe** | SPM + 전해질 확산. 농도 구배 해석. |
| Doyle-Fuller-Newman / Pseudo-2D | **DFN / P2D** | 두께(x) 1D + 입자(r) 1D = 의사 2D. 모든 변수 공간 의존. 고정밀. |
| 다입자 모델 / Multi-Particle Model | **MPM** | 입자 크기 분포 (PSD) 반영. bi-modal·Si/Gr blend. |
| 다종다반응 / Multi-Species Multi-Reaction | **MSMR** | 다단계 상전이 (Si, LFP). |
| 등가회로 모델 / Equivalent Circuit Model | **ECM** | R·C·전압원 회로 근사. (§8 상세) |
| 축약 모델 / Reduced Order Model | **ROM** | DFN → 저차 축약. BMS 실시간. |
| Multi-Scale Multi-Domain | **MSMD** | NREL 의 입자→셀→팩 결합 프레임. |

## 1.2 관련 배터리 용어 (부속)

- **Half cell** — 기준 전극(Li metal) 대비 단일 전극 측정. OCP·D_s 추출에 필수
- **Full cell** — 양·음극 결합 실제 셀
- **Jelly roll (JR)** — 권취형 셀 내부 적층 구조 (권심·권말·R 부)
- **Pouch / Can** — 외장. Pouch 는 laminate film, Can 은 Al/Steel
- **Multi-layer composite** — 양극·분리막·음극 적층

## 1.3 BDT / PyBaMM 적용

- BDT 는 SPM·SPMe·DFN 3종 지원 (`model_map`). MPM·MSMR 미지원 (Si/SiC blend → 향후 확장)
- 모델 선택 결정 트리: BMS·실시간 → SPM / DCIR·GITT → SPMe / 정밀 수명·열화 → DFN

→ [[PyBaMM_정리]], [[PyBaMM_Variables_PPT]], [[Silicon]] (Si blend → MPM 필요), [[MSMD]]

# 2. 셀 전압 분해 (Voltage Decomposition)

> **개념**: 시뮬의 가장 기본 출력. V_cell 을 구성 항으로 분해하면 어떤 손실이 지배적인지 보인다.

## 2.1 지배 식

$$V_{cell} = U_{pos}(x_{pos}) - U_{neg}(x_{neg}) - \eta_{pos} - \eta_{neg} - I \cdot R_{ohm}$$

## 2.2 시뮬 용어

| 한글 / 영문 | 약어 | 단위 | 정의 |
|----------|:----:|:----:|------|
| 단자 전압 / Terminal voltage | V_cell | V | 운용 중 측정되는 셀 전압 |
| 개방회로 전압 / Open-Circuit Voltage | **OCV** | V | 평형 시 V_cell = U_pos − U_neg |
| 개방회로 전위 / Open-Circuit Potential | **OCP** | V | 단일 전극의 평형 전위 (vs Li/Li⁺) |
| 과전압 / Overpotential | **η** | V | V_cell − OCV. 비가역 손실 |
| 반응 과전압 | η_rxn | V | Butler-Volmer kinetics 항 |
| 농도 과전압 | η_conc | V | 표면-벌크 농도차 항 |
| 옴 손실 | IR / V_ohm | V | 집전체·전해질·접촉 저항의 IR drop |
| Butler-Volmer 식 | **BV** | — | j = j₀ [exp(αFη/RT) − exp(−(1−α)Fη/RT)] |
| Tafel 근사 | — | — | 큰 η 에서 BV 한쪽만 — log j ∝ η |
| 전압 히스테리시스 / Hysteresis | — | mV | 충/방전 OCV 차. Si 150~300 mV |
| 전압 이완 / Voltage relaxation | — | — | Pulse 후 OCV 도달까지 시간. Si 수시간~수십시간 |

## 2.3 관련 배터리 용어 (부속)

- **양극 / 음극 / 전해질** — V_cell 의 세 구성 요소
- **상한·하한 전압 cut-off** — 충방전 종료 조건. NMC 표준 4.2V, **Gen5+ 고전압 LCO 4.53V**
- **전압 plateau (평탄 영역)** — 상변화 구간, dU/dx ≈ 0
- **dV/dQ 분석** — 미분 곡선으로 전극별 피크 식별 (LAM 진단)
- **첨가제** (FEC, LiDFOB 등) — η 와 SEI 안정성에 영향

## 2.4 측정 anchor

- **Half cell OCP 측정** → 시뮬 OCP 함수 입력 (4.53V 까지 측정 필수)
- **Three-Electrode (3전극)** → η_pos / η_neg 분리 실측 (Proto08, CA04 시트)

→ [[충방전_매커니즘]], [[양극별_특성]], [[Silicon]] (히스테리시스), [[260422_analysis_pybamm_key_parameters]] §6.6

# 3. Li⁺ 수송 / Rate-determining 단계

> **개념**: 충방전 시 Li⁺ 이 7단계를 거치며, **가장 느린 단계가 전체 속도 결정**. 시뮬은 각 단계의 시간상수 τ 를 분해해 보여준다.

## 3.1 7단계 경로 (직렬 결합)

```
① 양극 집전체 → 활물질 (전자 전도)        τ ~ μs~ms
② 양극 입자 내 Li 확산                    τ_p = r_p² / D_s,p   ← rate-limiting 후보
③ 양극 표면 반응 (BV)                    ∝ 1/j₀,p
④ 전해질 수송 (양극→분리막→음극)           τ_e = L² / D_e_eff
⑤ 음극 표면 반응 (BV)                    ∝ 1/j₀,n   ← Li plating 경쟁 지점
⑥ 음극 입자 내 Li 삽입                    τ_n = r_n² / D_s,n   ← 급속충전 한계
⑦ 음극 활물질 → 집전체                    빠름
```

## 3.2 시뮬 용어

| 용어 | 약어 | 단위 | 정의 |
|------|:----:|:----:|------|
| 고체상 확산계수 | **D_s** | m²/s | 활물질 입자 내 Li 확산. SOC 의존. NMC 10⁻¹⁰~10⁻¹², Gr 10⁻¹¹~10⁻¹³, **Si 10⁻¹⁴~10⁻¹⁶** |
| 전해질 확산계수 | **D_e** | m²/s | 전해질 내 Li⁺ 확산 |
| 전해질 전도도 | **κ_e** | S/m | 전해질 이온 전도. 보통 c, T 의 함수 |
| 입자 반경 | r | m | 활물질 입자 반경. D50/2 |
| Bruggeman 계수 | β | — | 다공성 매체 유효 수송 보정 (보통 1.5). τ_eff = ε^(1−β) |
| 양이온 수송수 | **t₊** | — | 전해질 내 Li⁺ 가 운반하는 전류 비율 (~0.2~0.4) |
| 열역학 인자 | — | — | 1 + d ln γ / d ln c |
| Fick 의 법칙 | — | — | 1st: J = −D∇c, 2nd: ∂c/∂t = D∇²c |
| Nernst-Planck | NP | — | 농도 + 전위 구배에 의한 이온 flux |
| Sand 방정식 | — | — | Δc_s = (2J₀/√(πD)) √t. Semi-infinite 확산 |
| Weppner-Huggins | **WH** | — | D_s = (4/9π)(r·dE/dt / dU/d√t)². GITT 핵심 |

## 3.3 관련 배터리 용어 (부속)

- **입자 D50 / bi-modal / 대립자·소립자** — D50 → r 변환 (가정 A2). bi-modal 은 MPM 필요
- **전해액 점도** — η ∝ 1/D_e
- **분리막 기공률** — ε_sep, Bruggeman τ_sep
- **합제밀도** — 활물질 체적분율 ε_AM ↔ 공극률 ε_pore. 고합제 = 고상 전도 ↑ + 전해질 수송 ↓
- **충방전 메커니즘** (주차타워 비유) — [[충방전_매커니즘]]

## 3.4 측정 anchor

- **GITT** → D_s(SOC) 직접 측정 (Weppner-Huggins). 가정 A2/A3 역검증
- **PITT (Potentiostatic ITT)** — GITT 의 voltage step 버전
- **EIS Warburg 영역** — D 추출 가능 (작도법)
- BDT 데이터셋: 240821 Gen4pGr / 250905 M2-SDI half / 251224 M1 ATL Half (양극·음극 별)

→ [[260419_GITT_확산계수_추출]], [[GITT]], [[합제밀도와_전류밀도]], [[충방전_매커니즘]]

# 4. 화학량·농도·용량 (Stoichiometry · Concentration · Capacity)

> **개념**: c_max·initial concentration·N/P ratio 가 셀 용량과 stoichiometry window 를 결정한다.

## 4.1 시뮬 용어

| 용어 | 약어 | 단위 | 정의 |
|------|:----:|:----:|------|
| 최대 농도 | **c_max** | mol/m³ | 격자에 들어갈 수 있는 Li 최대 농도. NMC ~63,000, **LCO ~51,500**, LFP ~22,000 |
| 화학양론비 | x, y | — | 활물질 격자 내 Li 비율 (0~1) |
| 리튬화 정도 | — | — | Stoichiometry 와 동일 또는 변환 |
| 충전 상태 / State of Charge | **SOC** | % | 셀 잔존 용량 비율 |
| 방전 깊이 / Depth of Discharge | **DOD** | % | 1 − SOC |
| 음극·양극 용량비 | **N/P ratio** | — | 보통 1.08~1.15. Li plating 안전 마진 |
| Stoichiometry window | Δx | — | 운용 중 x 변동 폭 |
| 초기 농도 | c_init | mol/m³ | Formation 후 정착된 Li 분포 |
| 비용량 | Q | mAh/g | 단위 질량당 용량 |
| Loading Level | **LL** | mg/cm² | 단위 면적 활물질 양 |

## 4.2 관련 배터리 용어 (부속)

- **양극 화학계별 c_max·V_cut**:
  - 표준 LCO 4.2V / **Gen5+ 고전압 LCO 4.53V** (deep delithiation x_p 0.3~1.0)
  - NMC622 / NMC811 (4.2V)
  - LFP 3.8V (2상 — A2 가정 파탄)
- **이론 용량 vs 실용 용량**: Si 이론 3579 mAh/g vs 실용 1500~2500 (crack 손실)
- **1st Coulombic Efficiency (ICE)** — 1st 사이클 방전/충전 비. 손실분이 초기 LLI
- **2nd reversible capacity** — Spec 2.4·2.5·3.4·3.5 필드. c_max 변환의 입력
- **Formation 효과** — SEI 초기 형성으로 N/P 정착
- **RPT** (Reference Performance Test) — 주기적 capacity 재측정

## 4.3 변환 anchor (가정 A3, A4)

- 비용량 → c_max: **c_max = Q × 3.6 × ρ / F × 10⁶** (가정 A3)
- N/P → initial concentration: stoichiometry window 매핑 (가정 A4)

# 5. 계면 반응 동역학 (Interfacial Kinetics)

> **개념**: Butler-Volmer 가 계면 반응 속도를 결정한다. j₀ 와 α 가 핵심 파라미터.

## 5.1 시뮬 용어

| 용어 | 약어 | 단위 | 정의 |
|------|:----:|:----:|------|
| 교환 전류 밀도 | **j₀** | A/m² | 평형 정·역 반응 속도 (BV 의 핵심) |
| 전하 이동 계수 | **α** | — | BV 비대칭 (보통 0.5) |
| 이중층 정전용량 | **C_dl** | F/m² | 계면 전하 저장 |
| 전하이동 저항 | **R_ct** | Ω | BV 의 선형 근사 저항 |
| 활성화 에너지 | **Ea** | J/mol | Arrhenius 의존 (k = A·exp(−Ea/RT)) |

## 5.2 관련 배터리 용어 (부속)

- **SEI / CEI** (Solid·Cathode Electrolyte Interphase) — 계면 보호막. 두께·저항이 j₀ 에 영향
- **첨가제** (FEC, VEC, PS, LiDFOB, LiPO₂F₂, HTCN) — SEI 안정성 → j₀ 와 R_ct 변경
- **활물질 표면 코팅** (예: Gen5+ 의 PF-PTO = Phosphate-based) — 양극 CEI + j₀ 동시 영향
- **도전재 네트워크** (SWCNT, Carbon black) — 전자 전도 → R_ct 보조

## 5.3 측정 anchor

- **EIS Nyquist 의 반원** → R_ct 직접 분리
- **HPPC** → Pulse 1초 후 voltage drop = R_ohm + R_ct
- **GITT pulse 첫 1~2초** → R_ct (이를 제외한 √t 영역만 D_s 회귀)
- **Cyclic voltammetry (CV)** → 첨가제 산화·환원 안정성 (CA04 시트)

→ [[전해액]], [[260422_analysis_pybamm_key_parameters]] §6.6 (가정 A6 첨가제)

# 6. 열화 모드 (Degradation Modes)

> **개념**: 수명 시뮬의 출력. 단일 "열화"가 아닌 **3 모드** 로 분리해 처리해야 한다.

## 6.1 시뮬 용어

| 용어 / 영문 | 약어 | 단위 | 정의 |
|----------|:----:|:----:|------|
| 리튬 인벤토리 손실 / Loss of Lithium Inventory | **LLI** | % | 사용 가능 Li 총량 감소 |
| 활물질 손실 (음극) / Loss of Active Material | **LAM_NE** | % | 음극 활물질 비활성화 |
| 활물질 손실 (양극) | **LAM_PE** | % | 양극 활물질 비활성화 |
| SEI 두께 / SEI thickness | — | m | 음극 SEI 성장량 |
| SEI 성장 속도 상수 | k_SEI | m/s | Kinetic constant |
| Lithium plating rate | k_pl | m/s | Plating 동역학 |
| Dead lithium | — | mol | Plating 후 비가역 격리 |
| 입자 균열 길이 | — | m | Particle cracking |
| Knee point | — | cycle | 수명 곡선 급락 시작점 |
| BDT 수명 모델 | — | — | Q(n) = 1 − a·n^b − c·exp(d·(n−e)) |

## 6.2 관련 배터리 용어 (부속)

- **Si 체적팽창 300%** → A1 (합제밀도) + A2 (단일 r) 동시 파탄. LAM_NE 가속
- **Si 80% 손실 vs Gr 10%** (Jiang 2022, 0~30% SOC 40℃) — LAM_NE 분리 필요
- **Co dissolution** — 고전압 LCO 특화 LAM_PE 메커니즘 (Proto08 ICP, CA04 시트)
- **SiO-SEI crust** (Choi 2025) — Si knee point 메커니즘
- **분해분석** (Tear Down) — MP106 / MP207 의 Critical / Major / Minor 분류
- **Gas swelling** — 가스 발생 → 두께 증가. PyBaMM 범위 밖
- **Floating test** — 만충 유지 → calendar aging 평가

## 6.3 PyBaMM 매핑

- OKane2022 프리셋 = SEI + Li plating 서브모델 활성화
- **Gen5+ 고전압 LCO**: LAM_pe rate constant 재튜닝 필요 (구조 변화 가속)
- **Si blend**: 가정 A9 (blend → 단일 음극) 적용 / Si ≥10% 시 MPM 전환

# 7. 발열 분리 (Heat Generation Separation)

> **개념**: Bernardi 식이 셀 발열을 3 항으로 분리. 시뮬은 각 항의 SOC·시간 의존성을 정량.

## 7.1 지배 식 (Bernardi 1985)

$$Q_{tot} = \underbrace{I \cdot \eta}_{\text{비가역}} + \underbrace{I \cdot T \cdot \frac{dU}{dT}}_{\text{가역}} + \underbrace{I^2 R}_{\text{Joule}}$$

## 7.2 시뮬 용어

| 용어 | 약어 | 단위 | 정의 |
|------|:----:|:----:|------|
| 총 발열 | Q_tot | W or W/m³ | 모든 항의 합 |
| 비가역 발열 | **Q_irr** | W/m³ | I·η. 과전압 손실의 발열 |
| 가역 발열 | **Q_rev** | W/m³ | I·T·(dU/dT). 엔트로피 변화. ± 가능 |
| 옴 발열 | **Q_ohm** | W/m³ | I²R. 집전체·전해질 IR |
| Joule 발열 | Q_J | W/m³ | Q_ohm 과 동일 |
| 엔트로피 계수 | **dU/dT** | V/K | SOC 의존. 측정 가능 (potentiometric) |
| 활물질 분해열 | — | J/mol | DSC 측정. 열폭주 trigger |

## 7.3 관련 배터리 용어 (부속)

- **셀 비열 C_p** — 보통 1000 J/kg·K (Gr/NMC 평균)
- **셀 밀도 ρ** — 보통 2500 kg/m³
- **dU/dT 측정** — Half cell 에서 SOC 별 측정 가능
- **고온 가속** — Arrhenius (Ea ~ 0.4~0.6 eV)

## 7.4 PyBaMM 출력

- `Volume-averaged Ohmic / Irreversible / Reversible heating [W.m-3]` — 시뮬 결과로 직접 분해
- `Volume-averaged total heating [W.m-3]` — 합산
- `Volume-averaged cell temperature [K/C]` — 결합 결과

→ [[260422_analysis_pybamm_key_parameters]] (Volume-averaged 변수)

# 8. 열 모델 + ECM 결합 (Thermal Modeling)

## 8.1 ECM (시뮬 용어)

| 용어 | 약어 | 정의 |
|------|:----:|------|
| 등가회로 모델 | **ECM** | 셀을 R·C·전압원 회로로 근사 |
| 1RC 모델 / Thevenin | — | R_ohm + (R_ct ∥ C_dl) + OCV |
| 2RC 모델 | — | + (R_diff ∥ C_diff) 추가 |

| 용어 / 영문 | 약어 | 정의 |
|----------|:----:|------|
| Lumped 열 모델 | — | 전 셀 단일 T(t). ODE 1개 |
| 1D 열 모델 | — | 두께 방향 분포. PyBaMM `thermal=x-full` |

| 비등방성 k | — | 두께 vs 면내 k 다름. Pouch 면내 ~10× |

- **HPPC** → R, dU/dT, BTMS 설계 입력

# 9. 열폭주 (Thermal Runaway)

> **개념**: 자가 발열 > 방열 → 폭주. 안전 시뮬의 trigger 와 propagation 평가.

## 9.1 시뮬 용어

| 용어 / 영문 | 약어 | 정의 |
|----------|:----:|------|
| 열폭주 / Thermal Runaway | **TR** | 셀 자가 발열 > 방열. >150℃ trigger |
| 열폭주 전파 / Thermal Runaway Propagation | **TRP** | 셀→인접 셀→팩 확산 |
| Newman-Tiedemann-Gu 모델 | **NTG** | 양극·SEI·음극 분해의 결합 발열 |
| Hatchard 모델 | — | TR Arrhenius 다단 분해 |
| 임계 온도 | T_crit | TR 진입 임계 |
| Self-heating onset | — | 자가 발열 시작 (ARC) |
| Vent / Exhaust | — | 가스 분출 단계 |
| Gas burst | — | 폭발적 분출 |

## 9.2 측정 기법 (시뮬-검증 anchor)

| 기법 | 약어 | 측정값 |
|------|:----:|------|
| Accelerating Rate Calorimetry | **ARC** | 단열 dT/dt vs T. 임계 온도 추출 |
| Differential Scanning Calorimetry | **DSC** | 가열 시 흡열·발열 곡선. 양극 분해 온도 |
| Thermo-Gravimetric Analysis | **TGA** | 가열 시 질량 변화 (분해·휘발) |

## 9.3 관련 배터리 용어 (부속)

- **양극 분해 온도** — DSC 측정. 표준 NMC ~200℃, 고전압 LCO 더 낮을 수 있음
- **SEI 분해** — ~80℃ 부터 시작
- **분리막 shrinkage** — ~120℃ (PE) / ~150℃ (PP) → 내부 단락 trigger
- **전해액 끓는점** — EC 248℃, EMC 110℃ 등 — 가스 분출 timing
- **Co dissolution** — LCO 특화. 고전압에서 Co²⁺ 용출 → LAM_pe + 음극 침적
- **Heat test 110℃-5hr** — CA04 DFMEA 시트

## 9.4 PyBaMM·시뮬 한계

- PyBaMM 1D 는 기본적으로 TR **부분 지원** (lumped thermal 의 ramping)
- ARC·DSC 데이터로 NTG 파라미터 피팅 → 별도 Python 모델 필요
- TRP 는 Multi-cell + 3D CFD 필요 (PyBaMM 범위 밖)

→ [[260422_report_cell_design_vs_pybamm]] §10 (Category D)

# 12. 수치 해석·도구 (Numerical & Tools)

> **개념**: PDE → 이산화 → 솔버 → 결과. 시뮬 워크플로우의 인프라.

## 12.1 이산화 (공간)

| 용어 / 영문 | 약어 | 정의 |
|----------|:----:|------|
| Differential-Algebraic Equation | **DAE** | 미분-대수 결합. 배터리 모델 일반형 (∂c/∂t = ∇·(D∇c) + 대수 BC·BV) |
| Ordinary Differential Equation | **ODE** | 시간만 의존. ECM·lumped thermal |
| Partial Differential Equation | **PDE** | 시간+공간. P2D, 1D thermal |
| Finite Volume Method | **FVM** | 보존형 이산화. PyBaMM 기본 |
| Finite Element Method | **FEM** | 약형 이산화. COMSOL·ANSYS |
| Finite Difference Method | **FDM** | 직접 차분. 1D 확산 단순 |
| Mesh / Discretisation | — | 공간 격자 + PDE 이산화 |
| `var_pts` (PyBaMM) | — | 영역별 격자 수 (x_n, x_s, x_p, r_n, r_p) |
| 격자 수렴성 / Mesh convergence | — | dx → 0 시 해 수렴 검증 |

## 12.2 시간 적분

| 용어 / 영문 | 약어 | 정의 |
|----------|:----:|------|
| 명시법 / Explicit | — | y_{n+1} = y_n + dt·f(y_n). 단순. CFL 안정 한계 |
| 암묵법 / Implicit | — | y_{n+1} = y_n + dt·f(y_{n+1}). 무조건 안정. NR 반복 |
| Forward Euler / Backward Euler | FE / BE | 1차 explicit / implicit |
| Crank-Nicolson | **CN** | 2차 implicit. (FE + BE)/2. 진동 가능성 (강한 비선형) |
| Runge-Kutta 4 | **RK4** | 4차 explicit ODE 표준 |
| Backward Differentiation Formula | **BDF** | 1~5차 implicit multistep. SUNDIALS IDA·CVODES 기본 |
| 적응 시간 적분 / Adaptive timestep | — | 잔차 기반 dt 자동 조정 |
| CFL 조건 | — | dt < dx/v. Explicit 안정 한계 |
| 수렴 허용오차 / Tolerance | atol, rtol | 절대 / 상대 오차. SUNDIALS 표준 옵션 |
| 잔차 / Residual | — | ‖F(x)‖ < tol → 수렴 판정 |
| Stiffness | — | 시상수 비 큰 ODE → implicit 필수 |

## 12.3 솔버 / 알고리즘

| 솔버 / 알고리즘 | 약어 | 정의 |
|--------------|:----:|------|
| **SUNDIALS** | — | DAE/ODE 솔버 라이브러리 (LLNL) |
| **IDA / IDAKLU** | — | SUNDIALS implicit DAE. PyBaMM 기본. K = KLU sparse |
| **CVODES** | — | SUNDIALS ODE + 민감도 |
| **CasADi** | — | Symbolic + AD + NLP. PyBaMM Expression Tree 백엔드 |
| **Newton-Raphson** | **NR** | x_{k+1} = x_k − J⁻¹·F. Implicit DAE·FEA 표준 반복 |
| **Picard / 고정점** | — | x_{k+1} = G(x_k). 단순 대입. NR 보다 느리지만 안정 |

| **KLU sparse solver** | KLU | Sparse LU 분해. IDAKLU 의 K. 고차원 P2D 시 필수 |
| **GMRES / BiCGSTAB** | — | Krylov 부분공간 반복법. 대규모 sparse 시스템 |
| **Jacobian** | J | ∂F/∂x. NR·implicit 핵심. AD 자동 계산 가능 |

# 14. 경험식 수명 모델 (Empirical Lifetime Models)

> **개념**: 물리 모델로 직접 풀기 어려운 수명을 stress factor 의 함수로 회귀. BMS·warranty·셀 사양 검증의 표준 도구. §6 (열화 모드) 와 상보적 — §6 은 메커니즘, §14 는 통계적 곡선.

## 14.1 지배 식 형태

| 형태 | 식 | 적용 |
|------|---|------|
| **Power law** | Q_loss = a · n^b | Cycle aging 표준 (b≈0.5 SEI, b≈1 wear-out) |
| **Square-root (parabolic)** | Q_loss = k · √t | Calendar aging, SEI 확산 제어 |
| **Arrhenius** | k = A · exp(−E_a/RT) | 온도 가속 |
| **Eyring** | k = A · T · exp(−E_a/RT) | 열·전압 결합 |
| **Tafel-like (V·DOD stress)** | k ∝ exp(αF·ΔV/RT) | 고전압·고DOD stress |
| **BDT 수명 모델** | Q(n) = 1 − a·n^b − c·exp(d·(n−e)) | knee 표현 (선형 + 비선형 가속) |
| **DOD 멱법칙** | k_DOD = (DOD/DOD_ref)^p | 깊은 방전 가속 |
| **Coupled stress** | k = k_T(T) · k_V(V) · k_DOD(DOD) | stress 곱 결합 표준 |

## 14.2 시뮬 용어

| 한글 / 영문 | 약어 | 단위 | 정의 |
|----------|:----:|:----:|------|
| 경험식 수명 모델 | — | — | physics-blind 회귀 모델 |
| 활성화 에너지 | **E_a** | eV or kJ/mol | Arrhenius slope. SEI 0.4 eV, plating 0.2 eV |
| 전-지수 / Pre-exponential | A | 단위 동일 | Arrhenius 계수 |
| 멱지수 / Exponent | b | — | n^b 또는 t^b. b=0.5 대표 |
| 칼렌더 수명 / Calendar aging | — | year | 휴지 시 자연 열화 (SOC, T 의존) |
| 사이클 수명 / Cycle aging | — | cycle | 충방전 반복 열화 |
| 등가 풀사이클 / Equivalent Full Cycle | **EFC** | — | ∫|I| dt / Q_nom |
| Stress factor | — | — | T, V, DOD, ΔSOC, C-rate 등 가속 변수 |
| Coupled stress 모델 | — | — | stress 곱·합 결합 |
| Knee onset | n_k | cycle | 비선형 가속 시작 |
| Capacity fade rate | — | %/cycle | dQ/dn (선형 영역) |
| Resistance growth rate | — | %/cycle | dR/dn |
| 절반 수명 / Half-life | — | cycle | Q = 0.5·Q_0 도달 |
| EOL (80%) / EOL (70%) | **EOL** | % | End-of-life 정의. 통상 80% (소비자), 70% (산업) |

## 14.3 관련 배터리 용어 (부속)

- **Severson 2019 (MIT-Stanford-TRI)** — 124 LFP cell, 100 cycle 데이터로 cycle-to-fail 예측 표준 벤치마크
- **NASA PCoE Battery Aging** — 18650 다중 셀 (B0005~B0007) calendar+cycle
- **CALCE (Maryland)** — 공개 cycle aging 데이터셋
- **Wang 2014 (graphite cycle)** — power law 의 표준 reference
- **Schmalstieg 2014 / Hess 2017** — calendar aging 함수형 reference
- **Floating test** — 만충 V 유지 → calendar aging 가속
- **Storage test** — 50% SOC, 60℃ 등 → calendar 분리

## 14.4 측정·분석 anchor

- **Coulomb counting** — ∫I dt → SOC·SOH 직접 추정 (drift 누적)
- **Incremental Capacity Analysis (ICA)** — dQ/dV 피크 → SOH·LAM 진단

## 16.4 관련 배터리 용어 (부속)

- **MIT-Stanford-TRI 데이터셋** (Severson 2019) — 124 LFP cell, knee 예측 표준 벤치마크

- **Toyota Research Institute (TRI)** — battery ML 선도 연구
- **Open Circuit Voltage curve fitting** — 휴지 OCV → SOH 추정

- **dQ/dV (ICA), dV/dQ (DVA)** — §14 참조. AI 입력 특성으로 활용
- **EIS feature extraction** — Nyquist·DRT 특성 → ML 입력
- **Pulse-based features** — HPPC 응답 → R_ohm·R_ct·τ → ML 입력

## 16.5 시뮬-AI 결합 워크플로우 anchor

- BDT cycle 데이터셋 → SOH 라벨 → LSTM/Transformer 학습 (RUL 예측)

- Multi-fidelity: SPM (빠름) + DFN (정밀) 결합 → 추정 정확도 ↑ 비용 ↓
- Digital Twin 단계: 실셀 측정 stream + PyBaMM 동기화 → BMS 적응형 보정
