# 전류 밀도
- 1C로 전류가 흐를 때 단위 면적에서 나오는 전류: 단위 면적당 용량
- 단위 면적당 용량: 활물질의 양에 비례하므로 동일 platform에서는 LL(Loading Level)과 비례
- 단면적(1cm2) 당 용량에 기여하는 활물질의 양에 의해 결정
- 보통 양극을 기준으로 전류밀도를 보는 이유는 음극의 경우 리튬 석출 때문
- 무조건 양극보다 더 큰 용량을 사용하기 때문에 (NP ratio) 양극 용량에 의해서 셀 용량이 좌우됨
	- ![[Pasted image 20251208121120.png]]
	- 활물질 1개가 1mAh라고 하면!
		- 활물질이 9개이므로, 1C 때 전류밀도 9mA/cm2 (용량밀도 9mAh/cm2)
		- 활물질이 18개 이므로, 1C 때 전류밀도 18mA/cm2 (용량밀도 18mAh/cm2)

1. 전체 전류(전체 용량)/ 반응면 면적(양극 전극 넓이)
	1.  1C 이므로 용량에 맞는 전류가 흐르게 되고 (4000mAh 셀이면 1C일 경우 4000mA가 흐릅니다.)
	2.  단위 면적 당이므로 전체 반응 면적을 생각해주면, (일반적으로 A, B면의 반응하는 부분의 단면적을 의미합니다.)
	3. 전류 밀도 (mA/cm2) = 용량 (mAh, typical) / 반응하는 부분의 전체면적 (cm2)/ (1h)
2. 단위 면적 기준 계산
	1. 1cm2에서 흐르는 전류를 구하며, LL 중에 바인더 도전제를 제외한 활물질만 용량에 기여하므로
	2. 동일 PF의 경우 활물질 비율, 활물질 용량이 동일하므로 전류밀도는 LL에 비례
	3. 전류 밀도 (mA/cm2) = LL (mg/cm2, typical) * 활물질 비율(%) * 활물질 용량 (mAh/g) / (1h)
# 합제 밀도
- 단위 부피당 들어가 있는 활물질의 양
- 누르면 두께가 줄어들게 되므로 단위 부피당 활물질 양은 늘어남
- •<span style="background:#affad1">합제 밀도(g/cc) = 전극 LL(mg/cm2) / 전극 두께(um)</span>
	![[Pasted image 20251208121345.png]]
	- 동일 활물질 LL에서 두께만 반으로 줄어든다면 합제는 2배로 증가
	- 전류밀도는 동일
	- 활물질 1개가 1mAh라고 하면 활물질 18개이므로 1C 때 18mA/cm2가 흐름 (용량 밀도는 18mAh/cm2)
