# Knee point

- Capacity Knee point 정의: IEEE Standard 485 TM 2020 "배터리 수명의 대부분 동안 용량이 천천히 감소하지만 후자의 단계에서 빠르게 감소하기 시작할 때"로 정의
- 정량적이지 않다.
- 수학적 정의: 곡률 계산 > 수명이 직선에서 가장 많이 이탈할 때.
- 데이터 노이즈가 장벽(충방전기, 챔버 온도 resolution)
