#### 회의 제목

SRIB weekly - RUL

#### 회의 일시

- 2025/09/30(화) 16:00 ~ 16:54 (Asia/Seoul GMT +09:00)

#### 참석자

<삼성전자>

- 선행Battery Lab.(MX) : 안성진 랩장/Safety Agent, 류성택
- Mobile Battery : Samarth Agarwal Head of Part, Herojit Sharma Laipubam, Rahul Kumar, Subramanian Swernath Brahmadathan

#### 전이 학습 및 학습 종료 기준 최적화

• 전이 학습 과정에서 200 사이클의 타겟 데이터에 대한 학습이 A2의 경향성을 따라가는 문제

- 타겟 데이터가 200 사이클만 제공되어 전체 데이터의 정확도에 영향을 미치는 현상

- 학습 종료 기준(epoch) 설정 필요성 논의

• 학습 종료 기준 최적화를 위한 방안

- MAE(Mean Absolute Error) 변화율 기반으로 학습 종료 기준 설정 검토

- 새로운 셀 타입에 대한 학습 종료 기준의 적용 가능성 검토 필요

• 전이 학습의 학습 종료 기준 설정 방안

- 학습 진행 상황(MAE 변화율 등)을 모니터링하여 최적의 학습 종료 기준 설정

- 새로운 셀 타입의 경우 기존 기준 적용이 어려울 수 있음

#### Stack 모델 및 Chain 모델 비교

• Stack 모델과 Chain 모델의 구조 및 차이점

- Stack 모델은 동일한 인코더를 사용하여 두 가지 예측(용량 및 프로 용량)을 수행

- Chain 모델은 두 단계의 학습을 통해 예측을 수행하며, 각 단계의 학습이 독립적일 수 있음

• Stack 모델의 장점

- 동일한 인코더를 사용하여 학습 효율성 향상

- 학습 과정에서 용량 및 프로 용량에 대한 정보를 동시에 활용 가능

• 향후 계획

- Stack 모델과 Chain 모델의 성능 비교를 위한 테스트 진행 예정

- 새로운 셀 타입에 대한 모델 적용 가능성 검토

#### 입력 특성 및 멀티피쳐 사용 관련 논의

• 입력 특성(제한 용량 및 R)의 사용에 따른 영향

- 제한 용량 및 R을 함께 사용할 경우 용량 상승 현상이 발생

- 멀티피쳐 사용의 장단점 및 적용 가능성 검토 필요

• 향후 계획

- 제한 용량 및 R을 함께 사용할 때의 성능 검증 진행 예정

- 새로운 셀 타입에 대한 입력 특성 최적화 방안 검토

#### 온도 변화 및 용량 예측 관련 분석

• 온도 변화가 용량 예측에 미치는 영향

- 메인 셀의 경우 200~300 사이클에서 온도 변화로 인한 용량 예측 오차 발생

- 서브 셀의 경우 평균 온도 변화는 작지만 용량 예측에 영향을 미치는 현상 관찰

• 향후 계획

- 온도 변화에 따른 용량 예측 오차 원인 분석 진행 예정

- 새로운 셀 타입에 대한 온도 변화 영향 검토

#### 학습 데이터 및 테스트 데이터 설정

• 학습 데이터와 테스트 데이터의 설정

- 학습 데이터는 SOP, 테스트 데이터는 A2로 설정하여 성능 검증

- 새로운 셀 타입에 대한 학습 및 테스트 데이터 설정 방안 검토 필요

• 향후 계획

- 새로운 셀 타입에 대한 학습 및 테스트 데이터 설정 최적화 진행 예정

#### 학습 종료 기준 설정 방안 검토

• 학습 종료 기준 설정 방안

- MAE 변화율을 기반으로 학습 종료 기준 설정 검토

- 새로운 셀 타입에 대한 학습 종료 기준 적용 가능성 검토

• 향후 계획

- 학습 종료 기준 설정을 위한 추가 테스트 및 분석 진행 예정
