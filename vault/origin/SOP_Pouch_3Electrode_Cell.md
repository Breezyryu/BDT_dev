# 파우치형 3전극 셀 제작 SOP (Dry Room)

## 1. 목적 (Objective)
본 문서는 **드라이룸(Dry Room) 환경**에서 [[LCO]] 양극, [[Graphite]] 음극, 리튬 메탈 기준 전극(Reference Electrode)을 사용하는 파우치형 3전극 셀의 제작 절차를 표준화합니다.

## 2. 필요 장비 및 재료

### 2.1 환경 및 재료
- **환경**: 이슬점 -40°C 이하 드라이룸
- **재료**:
  - 양극: [[LCO]] (15.0 mg/cm²)
  - 음극: 인조 [[Graphite]] (7.8 mg/cm²)
  - 기준 전극: [[Lithium Metal]] Foil (>99.9%, 300µm)
  - 분리막: [[Separator]] (PE, 16µm)
  - 전해액: 1.0M LiPF6 in EC/DEC (1:1) + VC 2%

### 2.2 장비
- **가공**: 전극 펀칭기, 톰슨 나이프
- **조립**: 초음파/스팟 용접기, 진공 실러, 글러브박스(옵션)
- **분석**: 충방전기 (3전극 채널 지원)

## 3. 제작 절차 (Procedure)

> [!WARNING] 안전 수칙
> 모든 공정은 드라이룸에서 진행하며, 개인 보호구(보안경, 니트릴 장갑, 방진복)를 착용하십시오.

### 3.1 전극 및 분리막 준비

- [ ] **1. 전극 펀칭**: 톰슨 나이프로 전극 절단.
    - *Note*: 음극은 양극보다 `1-2mm` 크게 제작 (Overhang 설계).
- [ ] **2. 건조**: 진공 오븐 건조.
    - 양극: 120°C, 음극: 80~100°C (12시간 이상)
- [ ] **3. 탭 부착**: 무지부에 탭 용접/테이핑.
    - 양극(Al) / 음극(Ni) / 기준전극(Ni)
- [ ] **4. 기준 전극 가공**: 리튬 포일을 5x10mm로 절단 후 니켈 탭 압착.
    - *Tip*: 리튬 표면 산화막(검은색)을 긁어내고 은백색 속살을 사용.
    - 기준 전극용 분리막으로 감싸 절연 처리.

### 3.2 셀 조립 (Stacking)

- [ ] **1. 하단 배치**: 파우치 필름(Deep-drawn) 위에 **양극** 배치 (활물질 위로).
- [ ] **2. 분리막**: 양극을 완전히 덮도록 분리막 배치.
- [ ] **3. 상단 배치**: **음극** 배치 (활물질 아래로, 양극과 대면).
- [ ] **4. 기준 전극 삽입**: 양/음극 스택 **측면**에 리튬 기준 전극 배치.
    - *Critical*: 기준 전극이 주 전극과 직접 닿지 않도록 주의 (단락 위험).
- [ ] **5. 고정**: 캡톤 테이프로 위치 고정.

### 3.3 패키징 및 주입

- [ ] **1. 1차 실링**: 주입구 1면을 제외한 3면 열융착 (180-200°C).
- [ ] **2. 전해액 주입**: 마이크로피펫으로 정량(약 3g) 주입.
- [ ] **3. 함침(Wetting)**: 30분 이상 대기 (진공 함침 권장).
- [ ] **4. 최종 실링**: 진공 실러 이용 (-90 kPa 이상) 완전 밀봉.

## 4. 후처리 및 평가

### 4.1 활성화 (Formation)
1. **Aging**: 상온 12시간 이상 휴지 (전해액 함침 안정화).
2. **Cycle 1**: 0.05~0.1C 저율 충방전 (SEI 형성).

### 4.2 3전극 측정 포인트
3전극 셀은 양극과 음극의 전위를 분리하여 측정합니다.
- **$V_{full}$**: 셀 전체 전압
- **$V_{cathode}$** (vs Li/Li+): 양극 거동 모니터링
- **$V_{anode}$** (vs Li/Li+): 음극 리튬 석출/동역학 모니터링
