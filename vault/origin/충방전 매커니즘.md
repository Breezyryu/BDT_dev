# 2. 급속 충전
## 급속 충전 고려
- 전자 이동 > 양극: Li이온 나옴 > 전해액 통과 > 음극: Li이온 들어감 > 전자 이동
- 전체 속도의 합이 전체 속도 좌우, 구간별 damage 발생 가능
- 음극이 받아주지 못한 Li 이온은 바깥쪽에 쌓이거나 부반응 발생(Li plating)
## 충전 속도
<span style="background:#d3f8b6">음극에 Li 이온이 들어가는 속도 한계에 따라 급속 충전의 한계가 결정 > 급속 충전 알고리즘 결정</span>
![[Pasted image 20251208115530.png]]

- 전자 이동: 전극 길이가 길수록 전자 이동 속도 저하
- 양극 Li 이온 나옴: 현재 수준에서는 damage 적음
- 전해액 통과: 전해액에서 Li이온의 전달 속도 낮음

### 주차타워에 비유
	- 낮은 양극 주차 타워에서 높은 음극 주차 타워로 이동하는 과정 (Li 이온의 이동)
	- 음극 주차장 자리: 1060개, 양극 주차장 자리: 1000개, 이동 차량 1000대
	- 빠르게 이동하는 방법과 주차 타워에 빠르게 잘 주차하는 방법이 필요
	- 충전 한계: 주차 타워에 빠르게 잘 주차하는 방법 (음극 충전 한계 증가)
	- 주차 시 사고가 나거나 못 들어가는 경우 – Li plating, Li loss 등
	- 주차 타워의 높이를 낮추고 (활물질 작게), 작은 주차 타워를 여러 개 세우고(전류밀도 낮게), 주차 타워의 입구를 늘리고(배향도 낮게), 입구 앞에 대기열을 만들고 (카본 코팅), 내부의 통로를 많이 만드는(합제 낮게) 등의 방법
	- 충전 속도: 빠르게 이동하는 방법 (저항 감소, 충전 속도 증가)
	- 주차장 이동하는 길을 넓히고 (전해액 전도도 향상, 전류 밀도 증가), 주차장 사이의 거리를 좁히는 등 (중간탭, 멀티탭..)의 방법
## 음극 전극 설계
- 설계별 장단점이 있으나 대체적으로 저전류밀도 concept 적용
 ![[Pasted image 20251208115031.png]]
### **전류밀도**
- 저전류밀도
	- 활물질의 양이 적을수록 반응 면적이 넓어짐
	- 활물질의 쌓이는 높이가 낮을수록 기재부터 활물질까지 거리가 감소됨
- 고전류밀도
	- 전극의 길이 감소로 전극의 저항 감소
### **합제밀도**
- 고합제 밀도 - 음극 전극의 전도도 향상, 활물질별 접촉 면적 넓어짐
- 저합제 밀도 - 전해액의 함침 특성 증가
### **기타**
- 도전제 함량, 바인더 변경, Double layer 등

## 충전 속도
### 셀 저항 감소
- 셀 저항은 물리적+화학적 저항으로 구성
- 물리적 저항: Tap 위치, 수량, 두께 변경
	![[Pasted image 20251208120120.png]]
- 화학적 저항: 소재 (저점도 전해액, 분리막), 전극 (바인더/도전제/전류밀도/ 합제) 변경 필요
### 급속충전 영향도
#### 셀 구조
- 급속 충전은 셀 내부의 위치별 산포 (권심, 전극 Edge, 두께 편차에 의한 압력 불균일) 등에 영향 받음
- 급속 충전 사용은 셀 간 산포도 증가하게 됨
	![[Pasted image 20251208120444.png]]
#### 셀 온도
- 온도에 따라서 내부 저항이 변경: 급속 충전에서는 그 영향이 더 커짐
- 저온
	- 저온에서는 화학 반응 속도가 느려짐 (저항 증가)
	- 음극에 Li 석출 가능성 증가
- 고온
	- 고온에서는 화학 반응 속도가 빨라짐 (저항 감소)
	- 저저항 음극으로 인해 만충 용량 증가
	- 양극 쪽에 damage 발생 가능성 높아짐
	- 전해액 분해 반응 가속화됨
#### Side effect
- 저장 특성 감소
	- 입자 사이즈 감소, 배향도 감소, Carbon 코팅
	- 전해액과 반응 영역 증가 – 저장 특성 감소
	- 전해액 점도 감소
	- 고온 시 휘발 특성 증가, 전해액 분해 증가
- 에너지 밀도 감소
	- 전류 밀도 감소: Ohmic 저항 특성 증가 (Tab과의 거리 멀어질 경우)
	- 저합제
	- 바인더, 도전제 함량 증가
- 고율 충전 시 저율 충전 대비 수명 특성 감소
	- 구조 불균일, 온도, 셀간 편차 등에 의한 영향도가 커짐
	- 고율 충전 시 Li 석출 발생 가능성 높음
## 전극 길이와 저항
- ==동일 전류 밀도에서 용량이 2배이면 면적이 2배가 되어 저항이 ½로 줄어야 하지만, 길이 방향으로 증가할 경우에는 기재 저항의 영향으로 저항의 감소가 덜되어 용량 대비 저항 감소가 적어지게 된다.==
- 추가로 단순화 시킨 저항이긴 하지만 실제로는 전기적 저항이 작아질 경우 Li 이온의 이동에도 도움이 되어

다른 요소의 저항에도 영향을 미칠 수 있다.
- Tap이 끝에 있다고 가정하면 다음과 같이 단순화
	- 전자, 리튬이온 경로![[Pasted image 20251208135936.png]]
- **전극 길이 증가**
	- 면적 2배 용량 2배
		![[Pasted image 20251208140822.png]]
	- Rn,Rp=10 / Re=100 이라 가정
		- 기존: Rn + Re +Rp = 120
		- 변경: Rn + Re | (Rn + Re + Rp) + Rp = 20 + 54.5
	- 용량은 2배가 되었지만 저항이 1/2로 줄지 않음
	- 충전은 C-rate로 진행하므로 용량 증가 2배 = 전류 2배
		- 전류 2배일 때 저항이 ½로 감소해야 동일 충전 profile로 보임
	- 추가적인 변수 (실제 tab 위치, 전극 상태 등등)들도 있음
- **전극 폭 증가**
	- 면적 2배 > 용량 2배
		- ![[Pasted image 20251208141322.png]]
			- 폭에 의한 저항은 1/2로 감소
			- 기존: Rn + Rs +Rp
			- 변경: Rn/2 + Rs/2 + Rp/2
			- 전류 2배일 때 저항이 ½로 감소해서 동일 충전 profile 수준을 보임
- Stack 병렬 연결
	![[Pasted image 20251208141746.png]]
	- 용량을 C, 양/음극 사이의 저항을 R이라고 하면 (Ohmic 저항 중 양/음극 사이의 저항만 산정)
	- Stack이 n개 일 때 (단면 병렬 연결로 가정)
		- 용량은 n * C
		- 저항은 병렬 연결이므로 R / n
	- 충전은 C-rate로 진행하므로, 용량 증가 n배 = 전류 n배
	- 전류 n배일 때 저항이 1/n로 감소해서 동일 충전 profile 수준을 보임

# 4. 기타 요인
- 함침별 차이
	- 함침이 잘된 부분의 경우 주변의 Li+과 전자의 결합 용이
	- 함침이 안될 경우 주변의 Li+이온 부족으로 표면의 Li+이온과 결합
- NP ratio 차이 or gap의 차이
	- 받아줄 음극이 적을 경우 Li 석출 발생
	- 다른 부분보다 거리가 멀 경우 과전압 걸려 Li 석출 용이
		- J/R의 R부에서 발생 가능
		- Overhang 부위에서 역발생 가능
		- 국부적인 deform 부위에서 발생 가능 (C-rate 영향에 따라 더 커짐)
- 국부적 온도에 의한 편차
	- 온도 편차에 의한 국부적 열화 발생
- 양극+ 열화
	- 급속 충전용 양극 사용으로 양극 전압 상승
	- 용량이나 충전 전압, 종단 C-rate로 제어

# 6. 급속 충전 개발 Process
- 개략도
	![[Pasted image 20251208142230.png]]
		*TA: Travel Adapter[[00_Inbox/!용어]]
	- 목표 1
		- 소구점 고려한 급속 충전 목표 설정 (마케팅, PSG와 협의)
		- SET 기준 10분 충전량 46%, 30분 충전량 85%, 만충 시간 53분 이내 고려
			![[Pasted image 20251208142643.png]]
	- 목표 2
		- TA(45W)와 용량(3500mAh 이상) 고려하여 2.8C/ 충전 조건 선정
			![[Pasted image 20251208142718.png]]
	- 알고리즘 결정
		- **목표: 10분 43%, 30분 85%, 만충시간 55분**
			- 가용 C-rate: 2.8C~1.4C base로 검토 (4:3:2)
			- 3 Step 선정: 2.8C, 2.1C, 1.4C
			- 마지막 Step 1.0C로 변경
				- PCM을 고려한 Pack max 전압 고려 (Pack 전압 기준 4.4V 넘지 않을 것)
				- 만충 전압 부근에서의 양극/전해액 열화 방지 (고율 4.4V 진행 시 양극/전해액 damage 감소)
				- 추후 FFC 고려 시 저율/고전압으로 진행 필요
				- 필요 시 2.1C와 1.0C 사이에 1.5C step 추가 가능
			- 최종 step 2.8C/ 2.1C/ 1.0C로 결정
				- 2.8C * 5min + 2.45C * 5min(CV, 0.7C 5min 지속) + 2.1C * 5min + 1.55C * 8min(CV) + 1.0C * 5min (90% 제한) + 0.45C * 2 min
				- = 10분 43%, 30분 92% 수준 만족, 만충 시간은 Cut-off C-rate로 결정 or FFC 적용
					![[Pasted image 20251208142947.png]]
	- **고려사항 - 발열**
		- 발열 예측
			- 2.8C 진행 시 max 전류 9A 기준으로 D2와 동일 전류 인가 예상
			- Cell 단독보다는 PCM부에서 발열 상승 예측
			- 충방전 알고리즘 기준으로 전류 인가하여 온도 상승 확인 (PSG 문의)
			- MAX current * 시간 (min)을 기준으로 산정, ex) 9A * 5분 이내 등
			- SET 기준 표면 기준 38도 이하 필요, Cell 표면 온도 확인 필요

- 고려사항
	- 업체별
		- 업체별 충전 속도 차이 존재 (업체별 셀 고유 특성)
		- 동일 충전 조건에서 충전량 높을 시 Cell에는 수명 가혹 조건
			1) ID pin 삽입, ID pin에 따른 별도 알고리즘 적용 (Step별 전압 상이)
			2) 신규 알고리즘 적용 (SOC cut-off, SOC에 맞춘 time cut-off 반영)
			3) Smart IC 적용(S12 이후) 및 별도 알고리즘 적용 (Step별 전압 상이)
			4) CC-CV step, CC step으로 변경 (발열 측면, 제어 측면 등)
			![[Pasted image 20251208171454.png]]
	- 세트-셀 차이
		1) 실 사용 조건에서 발열 제어 감소를 위하여 CC-CV step을 CC step으로 변경
